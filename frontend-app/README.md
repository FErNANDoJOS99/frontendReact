

## Antes de ejecutar

Asegúrate de tener instalado:

- **Node.js** (versión 18 o superior recomendada)
- **npm** (incluido con Node.js)
- **Backend API** ejecutándose en `http://localhost:8000`

## Cómo Ejecutar el Proyecto



```bash
# 1. Navegar al directorio de react
cd frontend-app

# 2. Instalar dependencias
npm install

# 3. Ejecutar aplicación
npm start
```


- Abre automáticamente [http://localhost:3000](http://localhost:3000) en tu navegador

### Ejecutar tests

```bash
# Ejecutar todos los tests
npm test

# Ejecutar tests sin modo watch
npm test -- --watchAll=false

# Ejecutar tests con información detallada
npm test -- --watchAll=false --verbose
```
